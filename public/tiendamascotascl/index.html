Acerca de TiendaMascotasCL

TiendaMascotasCL v1.0

Sistema de gestión integral para tiendas de mascotas en Chile.

Características:

  • Gestión de inventario con alertas de stock
  • Sistema POS con múltiples métodos de pago
  • Base de datos de clientes con historial de compras
  • Agenda de grooming profesional
  • Gestión de proveedores
  • Reportes analíticos detallados

Datos almacenados localmente en tu dispositivo con localStorage.

Todos los datos se guardan automáticamente. No requiere conexión a internet después de la instalación inicial.
