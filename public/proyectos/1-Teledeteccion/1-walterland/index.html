Detalles Técnicos

Este análisis se basa en un procesamiento de datos geoespaciales para extraer indicadores biofísicos clave a partir de imágenes satelitales.

  • Fuente de Datos: Se utilizó la colección de imágenes Sentinel-2 (COPERNICUS/S2_SR_HARMONIZED), que proporciona datos de reflectancia superficial con una resolución de hasta 10 metros.

  • Preprocesamiento: Se aplicó un filtro de nubes utilizando la banda de clasificación de escena (SCL) para garantizar la calidad de los datos, descartando píxeles afectados por nubes o sombras.

  • Índice NDVI: Se calculó el Índice de Vegetación de Diferencia Normalizada (NDVI) con la fórmula: (B8 - B4) / (B8 + B4). Este índice mide el vigor y la densidad de la vegetación.

  • Índice NDWI: Se calculó el Índice de Agua de Diferencia Normalizada (NDWI) usando la fórmula: (B3 - B8) / (B3 + B8) para evaluar el contenido de humedad en la vegetación.

  • Análisis de Series de Tiempo: Se procesaron imágenes desde 2017 hasta la fecha para generar un gráfico y una tabla de datos que muestran la evolución promedio de NDVI y NDWI en el polígono de interés.
