Framework CUSSEN-VolcanicActivity v2.0

El Super Índice CUSSEN es un índice compuesto propietario diseñado específicamente para el monitoreo integrado de actividad volcánica. Combina cinco componentes espectrales y térmicos para generar una métrica unificada de actividad volcánica.

Fórmula del Super Índice

SuperIndex = (w₁ × Thermal) + (w₂ × Plume) + (w₃ × Ash) + (w₄ × Vegetation) + (w₅ × Surface) Donde: w₁ = 0.30 (Componente Térmico) w₂ = 0.25 (Pluma Volcánica) w₃ = 0.20 (Cenizas/Aerosoles) w₄ = 0.15 (Vegetación/Impacto) w₅ = 0.10 (Cambios Superficiales)

Metodología y Stack Tecnológico

Fuentes de Datos

  • MODIS Terra/Aqua: Datos térmicos (Bandas 31, 32) y VRP (Volcanic Radiative Power)
  • Sentinel-5P TROPOMI: SO₂, CO, aerosoles volcánicos
  • Sentinel-2 MSI: Índices espectrales, cambios superficiales
  • Landsat 8/9 OLI: Análisis complementario multi-espectral

Procesamiento

Google Earth Engine: Procesamiento en la nube, agregación temporal, cálculo de índices compuestos

R (Tidyverse + sf + terra): Análisis estadístico avanzado, detección de cambios, series temporales, Machine Learning

Técnicas aplicadas: Changepoint detection, ARIMA, Random Forest, correlación cruzada, análisis espectral

Scripts Desarrollados

  • CUSSEN_Kilauea_SuperIndex_GEE.js - Script GEE para cálculo del Super Índice
  • CUSSEN_Kilauea_Analysis_R.R - Análisis estadístico integrado en R
  • CUSSEN_Process_GEE_Data.R - Pipeline de procesamiento de datos

Hallazgos Principales

  • Actividad Moderada Sostenida: El Super Índice promedio de 0.244 indica actividad volcánica moderada durante el período analizado
  • 3 Episodios Eruptivos: Detectados mediante análisis de changepoint en la serie temporal
  • Componente Térmico Dominante: La señal térmica (0.668) fue el componente más consistente, indicando persistencia de actividad magmática
  • Pluma Volcánica Variable: Alta desviación estándar (0.116) en el componente de pluma, sugiriendo eventos esporádicos de emisión
  • Impacto Vegetativo Bajo: Cambios en vegetación (0.134) indican impacto ambiental limitado en el período analizado
