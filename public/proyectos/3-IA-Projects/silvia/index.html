Descripción

SILVIA - Asistente Inteligente de Alimentación

App Flutter que combina gestión de inventario, generación de recetas con OpenAI GPT-4, planificación de menús semanales y scraper de precios de supermercados chilenos.

Features REALES (del README)

Inventario Inteligente
Recetas con IA
Menú Semanal
OCR Productos
Scraper Precios Supermercados
Lista Compras Auto

Supermercados Integrados: Scraper automático de precios de principales cadenas chilenas (Lider, Jumbo, Tottus, etc.)

Stack Tecnologico

  • Flutter - Framework multiplataforma

  • Firebase - Backend & Database

  • OpenAI GPT-4 - Generación de recetas

  • Tesseract OCR - Escaneo de productos

  • Python Flask - Backend services
