CUSSEN Fire Monitor v3.0

Sistema profesional de monitoreo de incendios forestales con detección de plumas de humo, análisis de severidad y predicción de propagación en tiempo real.

PRODUCCIÓN

Descripción

DEPLOYED - Sistema operativo desarrollado para la emergencia de Concepcion 2025

CUSSEN Fire Monitor implementa el Framework CUSSEN v3.0 de índices espectrales adaptativos con integración directa a Google Earth Engine. Monitorea incendios forestales en tiempo real mediante Sentinel-2, con análisis de severidad (dNBR + CUSSEN-Incendio v2.0), detección de frentes activos via SWIR y mapeo de dispersión de plumas de humo.

  • App Shiny desplegada: Ver App Live
  • Framework CUSSEN-Incendio v2.0 optimizado para bosque nativo chileno
  • Detección de plumas de humo (CUSSEN-Plume)
  • Integración con datos meteorológicos NOAA/GFS

Framework CUSSEN-Incendio v2.0

Formula: IC = (w_nir*NIR - w_swir*SWIR2 - w_red*Red) / (w_nir*NIR + w_swir*SWIR2 + w_red*Red)

Pesos adaptativos: w_nir=1.2 (vegetación viva) | w_swir=1.5 (area quemada) | w_red=0.8 (ceniza)

Optimizado para bosque nativo chileno con calibración por Machine Learning.

Clasificación de Severidad (USGS/BAER)

dNBR < -0.1: Rebrote Alto

-0.1 a 0.1: Sin Quemar

0.1 a 0.27: Severidad Baja

0.27 a 0.44: Severidad Moderada-Baja

0.44 a 0.66: Severidad Moderada-Alta

> 0.66: Severidad Alta
