Ahorcado Futurista

El clasico juego del ahorcado reimaginado con estetica cyberpunk, efectos neon, sonido sintetizado y musica de fondo generativa.

PRODUCCION

Descripción

DEPLOYED - Juego interactivo desplegado en ShinyApps.io

Ahorcado Futurista es una reimaginacion completa del clasico juego del ahorcado con estetica cyberpunk/neon. Desarrollado en R Shiny con audio sintetizado en tiempo real usando Web Audio API, musica de fondo generativa, efectos visuales animados y diseño mobile-first optimizado para dispositivos tactiles.

  • Desplegado: Jugar Ahora
  • Audio sintetizado con Web Audio API (sin archivos de sonido)
  • Musica de fondo generativa con osciladores
  • Diseño mobile-first con teclado virtual
  • Animaciones CSS cyberpunk (glitch, neon glow, scanlines)

Caracteristicas

Audio Sintetizado

Web Audio API en tiempo real

Teclado Virtual

Optimizado para tactil

Efectos Cyberpunk

Glitch, neon, scanlines

Gamificacion

Puntos, rachas, niveles

Sistema de Audio

Letra correcta: Acorde ascendente C5→E5→G5 (sine wave)

Letra incorrecta: Tono descendente 200→100 Hz (sawtooth wave)

Victoria: Fanfarria C5→E5→G5→C6

Derrota: Descenso cromatico dramatico

Musica de fondo: Osciladores generativos con pads atmosfericos

Stack Tecnologico

  • R Shiny - Framework reactivo

  • Web Audio API - Sintesis de sonido

  • CSS3 Animations - Efectos cyberpunk

  • Mobile-First - Teclado virtual tactil

  • ShinyApps.io - Hosting
