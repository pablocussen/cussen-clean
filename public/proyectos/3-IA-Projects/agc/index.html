AGC

Auditoría Geoespacial de Carbono - Certificación de proyectos forestales con satélites.

EN DESARROLLO

Descripción

Sistema de auditoría geoespacial para certificación de proyectos de captura de carbono. Utiliza análisis satelital multitemporal e IA para verificar biomasa forestal y calcular créditos de carbono.

Caracteristicas

Monitoreo Satelital
Cálculo Biomasa
Reportes Certificación
Detección Cambios

Stack Tecnologico

  • Google Earth Engine - Análisis satelital

  • Python - Procesamiento datos

  • ML Models - Estimación biomasa

  • QGIS - Análisis geoespacial

  • Pandas/NumPy - Data analytics
