KiteWeather Pro

Recomendación de tamaño cometa con ML. Flutter + Weather APIs (OpenWeather, Windy, Stormglass) + ML Analytics + Firebase.

EN DESARROLLO

Descripción

Aplicación móvil especializada en pronóstico meteorológico para kitesurf. Recomendaciones inteligentes de tamaño de cometa, mejores horarios de sesión y análisis de tendencias climáticas.

Caracteristicas

Pronóstico 7 Días
Recomendación Cometa
Análisis Spots
Alertas Push

Stack Tecnologico

  • Flutter - Framework multiplataforma

  • Weather APIs - OpenWeather, Windy, Stormglass

  • ML Analytics - Predicción de condiciones

  • Firebase - Backend & Database
