Arkitecto AI

Análisis inteligente de presupuestos de construcción usando IA

PRODUCCIÓN

Descripción

PWA que utiliza Vertex AI (Gemini 1.5 Pro) para analizar imágenes de proyectos de construcción (planos, fotos, bocetos) y generar presupuestos estimados basados en base de datos de APUs (Análisis de Precios Unitarios).

Base de datos: 347 APUs profesionales chilenas desde archivos Excel

Producción: https://arkitecto-ai.vercel.app/

Core Features

Proceso Critico (Setup)

1. Ingerir APUs: python ingest_apus.py (347 items a Firestore)

2. Backend: python main.py (puerto 8000)

3. Frontend: npm run dev (puerto 5173)

Stack Tecnologico

  • Python FastAPI 3.11+ - Backend

  • React 19 + TypeScript - Frontend

  • Vite - Build tool

  • Vertex AI Gemini 1.5 Pro - IA

  • Firebase Firestore - Database

  • TailwindCSS - Styling

  • Framer Motion - Animaciones

  • vite-plugin-pwa - PWA

  • Vercel - Deployment
