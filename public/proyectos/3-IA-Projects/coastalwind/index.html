CoastalWind.AI

Sistema de predicción de viento costero con IA (96% precisión)

PRODUCCIÓN

Descripción

Sistema de predicción de viento costero usando Machine Learning (XGBoost) que resuelve el problema de pronósticos inexactos por bloqueo térmico en zonas costeras.

Métrica Principal:

R² = 0.959 (95.9% de precisión en predicciones)

Modelo XGBoost entrenado con datos históricos de temperatura superficial del mar (MODIS) y variables meteorológicas vía Google Earth Engine.

Caracteristicas

Predicción ML 96%
Análisis Térmico
Datos Satelitales MODIS
API REST
Dashboard Interactivo

Stack Tecnologico

  • Python FastAPI - Backend API

  • XGBoost - Modelo ML

  • Google Earth Engine - Datos satelitales

  • Streamlit - Dashboard

  • Docker - Deployment
