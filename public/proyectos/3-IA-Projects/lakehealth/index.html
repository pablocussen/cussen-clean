CUSSEN-LakeHealth v2.0

Monitor de salud ambiental del Lago Llanquihue. Teledetección satelital para fiscalización de calidad de agua con índices CUSSEN adaptativos.

PRODUCCIÓN

Descripción

DEPLOYED - Desarrollado para el Servicio de Salud Los Lagos

CUSSEN-LakeHealth es un sistema profesional de monitoreo de calidad de agua para el Lago Llanquihue, utilizando el CUSSEN Framework v2.0 de índices espectrales adaptativos. Analiza imágenes Sentinel-2 MSI para detectar eutrofización, cianobacterias, turbidez y materia orgánica. Diseñado para apoyar la fiscalización ambiental y la toma de decisiones en salud pública.

  • Dashboard Shiny interactivo con mapas Leaflet
  • CUSSEN-LakeHealth SuperIndex (CLH) con 5 componentes
  • Clasificación automática del estado trófico
  • Sistema de alertas por zona
  • Generación de reportes exportables

CUSSEN-LakeHealth SuperIndex

Formula: CLH = w1*CEI + w2*CTI + w3*CCI + w4*COI + w5*CCL

CEI Eutrophication Index - Clorofila-a, fitoplancton
CTI Turbidity Index - Turbidez, sedimentos
CCI Cyanobacteria Index - Cianobacterias, ficocianina
COI Organic Matter Index - CDOM, materia orgánica
CCL Clarity Index - Transparencia (inverso)

Automáticas por CLH

Clasificación CLH Score

0 - 20 Excelente: Vigilancia rutinaria

20 - 40 Buena: Monitoreo estándar

40 - 60 Moderada: Aumentar frecuencia de muestreo

60 - 80 Degradada: Investigación activa + fiscalización

80 - 100 Crítica: Acción inmediata + cierre preventivo
