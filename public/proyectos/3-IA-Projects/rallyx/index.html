RallyX

Simulación y Análisis de Rally Deportivo

PRODUCCIÓN

Problema → Solucion

Problema

Organizadores de rally necesitan planificar rutas, calcular tiempos y gestionar eventos de forma eficiente.

Solución

Aplicación web con Google Maps API para waypoints, análisis de tiempos por tramo y exportación KML/GPX.

Caracteristicas Principales

Planificación Rutas
Sistema de waypoints interactivo
Análisis Tiempos
Cálculo automático por tramo
GPS Tracking
Seguimiento en tiempo real
Export KML
Compatible con Google Earth

Stack Tecnologico

  • HTML5

  • JavaScript

  • Maps API

  • PWA

  • GitHub Pages
