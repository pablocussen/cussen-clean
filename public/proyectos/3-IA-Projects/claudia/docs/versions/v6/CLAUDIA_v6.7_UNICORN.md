1. Presiona `Ctrl+K`
2. Escribe lo que buscas ("exportar", "precios", "tareas")
3. Navega con flechas ↑↓
4. Enter para ejecutar

### AI Suggestions
1. Usa CLAUDIA normalmente
2. El sistema aprende tu comportamiento
3. Recibirás sugerencias contextuales cada 5-10 min
4. Acepta o rechaza sugerencias
5. Desactiva con `localStorage.setItem('claudia_ai_suggestions', 'false')`

### Export PRO
1. Click en botón "Exportar"
2. Elige formato (PDF, Excel, Word, etc.)
3. Selecciona contenido a incluir
4. Elige plantilla
5. Descarga automática

### Performance Monitoring
1. Abre consola del navegador
2. Busca "📊 Performance Metrics"
3. Ve badge en esquina inferior derecha
4. Click para ver métricas detalladas
5. Historial en `localStorage.getItem('claudia_perf_history')`

## 🔧 Build & Deploy

### Local Development
```bash
# Install dependencies
cd scripts
npm install

# Build bundle
npm run build

# Minify
npm run minify:js
npm run minify:lazy
npm run minify:css

# Full build
npm run full-build

# Analyze bundle
npm run analyze
```

### Production Deploy
```bash
# Deploy to Firebase
firebase deploy --only hosting

# Logs
firebase hosting:channel:list
```

### Environment
- **Dev:** `localhost:5000`
- **Staging:** Firebase preview channels
