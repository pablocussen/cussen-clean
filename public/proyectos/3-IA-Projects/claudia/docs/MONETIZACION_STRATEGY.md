**Ubicación:** Botón "Ahorrar +15%"
**Valor:** Muestra ROI tangible del upgrade

**Ubicación:** Nuevo botón "Comparar Precios"
**Valor:** Feature killer, ahorro real

## 📈 Estrategia de Conversión

### Funnel de Conversión

```
👤 Visitante
    ↓ (Landing page con value prop)
📝 Registro FREE
    ↓ (Onboarding + primer proyecto)
🎯 Usuario Activo FREE
    ↓ (Usa features, ve valor, llega a límites)
💳 Upgrade a PRO
    ↓ (Uso avanzado, ahorro real)
⭐ Usuario PRO Recurrente
```

### Triggers de Conversión

**Momento Ideal para Mostrar Upgrade:**
1. **Al alcanzar límite** (3 proyectos, 15 actividades)
2. **Después de calcular ahorro** (Dashboard muestra $100K+ ahorro potencial)
3. **Al intentar comparar mayoristas** (Feature bloqueado)
4. **Después de 5 exports** (Límite mensual)
5. **Proyecto grande** (>$5M CLP, ahorro justifica PRO)

**Mensajes Clave:**
- "Ahorra $127.450 en este proyecto con PRO"
- "El PRO se paga solo con 1 proyecto"
- "Usuarios PRO ahorran 18% en promedio"
