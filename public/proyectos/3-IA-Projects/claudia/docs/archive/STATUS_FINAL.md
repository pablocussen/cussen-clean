### Stack
- **Frontend:** Vanilla JS + CSS3 + HTML5
- **Build:** Terser + CSSO + Custom bundler
- **Backend:** Python 3.11 + Google Cloud Functions
- **Hosting:** Firebase Hosting
- **PWA:** Service Worker + IndexedDB
- **APIs:** Telegram Bot API + Web Scraping

### Performance Optimizations
1. ✅ Aggressive minification (3 passes)
2. ✅ Tree shaking
3. ✅ Code splitting
4. ✅ Lazy loading
5. ✅ Smart prefetching
6. ✅ Cache strategies
7. ✅ Resource hints
8. ✅ Critical CSS inline

```bash
# Build bundle
cd scripts && npm run build

# Minify
npm run minify:js
npm run minify:lazy

# Analyze
npm run analyze

# Full build
npm run full-build

# Deploy
firebase deploy --only hosting
```
