                        DEPLOYMENT INSTRUCTIONS
================================================================================

QUICK START:
1. Copy all files to /c/temp/firebase-deploy/public/tapizerocl/
2. Serve over HTTPS (required for PWA)
3. Access at https://your-domain.com/tapizerocl/
4. App is immediately installable

FIREBASE HOSTING:
1. firebase init hosting
2. Set public directory to firebase-deploy/public
3. firebase deploy
4. App live at firebase-app.web.app/tapizerocl/

STATIC SERVER:
1. Serve /tapizerocl/ directory
2. Use HTTPS (localhost OK for dev)
3. Configure MIME types for manifest.json (application/json)
4. SPA routing: all requests → index.html (optional for hash routing)

DOCKER:
```dockerfile
FROM nginx:alpine
COPY firebase-deploy/public/tapizerocl/ /usr/share/nginx/html/
EXPOSE 443
```
