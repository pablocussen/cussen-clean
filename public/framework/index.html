VOL.01 // CTAI
Anomalia Termica
Deteccion de anomalias termicas en flujos de lava activos y domos volcanicos usando bandas SWIR.
CTAI = (w_b12 x B12 + w_b11 x B11 + w_b8a x B8A + w_b4 x B4) / (B12 + B11 + B8A + B4) w_b12 = 2.5 SWIR2 — maxima sensibilidad termica w_b11 = 1.8 SWIR1 w_b8a = -1.2 NIR — contraste negativo w_b4 = -0.5 Rojo — correccion atmosferica
Metricas: Temperatura de brillo (T = B12 x 4800 + 273K), Poder Radiativo Volcanico (VRP en MW)
Kilauea
VOL.02 // CVPI
Dispersion de Penacho
Deteccion de penachos volcanicos (SO2, aerosoles) y estimacion de altura de columna eruptiva.
CVPI = (w_aer x B1 + w_blue x B2 + w_green x B3 + w_nir x B8A) / (B1 + B2 + B3 + B8A) w_aerosol = 2.0 Banda costera — sensible a SO2 w_blue = 1.5 Azul w_green = -1.0 Verde — negativo w_nir = -1.5 NIR estrecho — negativo
Metricas: Indice Aerosol Volcanico (VAI), Altura de penacho (h = CVPI x 5000 + 1000 m)
Kilauea
VOL.03 // CVAI
Deposicion de Ceniza
Mapeo de depositos de ceniza volcanica y estimacion de espesor post-eruptivo.
CVAI = (w_vis x (B2+B3+B4)/3 + w_swir x (B11+B12)/2) / (B2+B3+B4+B11+B12) w_vis = 1.0 Visible — alta reflectancia de ceniza w_swir = -2.0 SWIR — baja reflectancia de ceniza
Metricas: Espesor estimado (thickness_mm = CVAI x 50 + 5 mm)
Kilauea
VOL.04 // CVII
Impacto en Vegetacion
Cuantificacion del impacto volcanico en ecosistemas vegetales: estres, mortalidad y recuperacion.
CVII = w_ndvi x NDVI + w_ndmi x NDMI + w_ndre x NDRE NDVI = (B8 - B4) / (B8 + B4) Vitalidad NDMI = (B8 - B11) / (B8 + B11) Humedad NDRE = (B8 - B5) / (B8 + B5) Red Edge w_ndvi = 0.4 w_ndmi = 0.3 w_ndre = 0.3
Kilauea Calbuco
VOL.05 // CSAI
Alteracion Superficial
Deteccion de cambios por flujos de lava, colapsos de crater y modificacion geomorfologica.
CSAI = (w_albedo x albedo + w_thermal x thermal_ratio) / 3.5 albedo = (B2 + B3 + B4) / 3 thermal_ratio = B12 / B11 w_albedo = 1.5 w_thermal = 2.0
Kilauea
VOL.06 // IMPACTO
Impacto Volcanico Ponderado
Cuantificacion multi-indice del impacto total de una erupcion: vegetacion, ceniza y cuerpos de agua.
Impacto = (w_dNBR x dNBR) + (w_dNDWI x dNDWI) + (w_dNDVI x dNDVI) Pesos ajustados por ML para maximizar deteccion de ceniza y lahares
Calbuco
FIR.01 // IC
Incendio v2.0 — Severidad de Quema
Deteccion de alta precision de severidad de incendios forestales y cuantificacion de biomasa residual. Modelo tri-banda con coeficientes adaptativos por tipo de bosque.
IC_v2 = ((w_nir x NIR - w_swir x SWIR2 - w_red x Red) / (w_nir x NIR + w_swir x SWIR2 + w_red x Red)) Los pesos (w) son calibrados por Random Forest segun tipo de bosque: coniferas, matorral, bosque nativo, plantacion
Ventaja: Supera al dNBR estandar al incorporar la banda Roja como corrector de humedad atmosferica.
Post-Incendio Dixie Fire Villa Alemana
AGR.01 // IPP
Productividad Agricola v2.0
Monitoreo de alta sensibilidad del impacto de enmiendas de suelo (biochar) en la productividad. Amplifica senal de clorofila restando ruido del suelo.
IPP = (w_evi x EVI) + (w_savi x SAVI) - (w_noise x BSI) EVI — Enhanced Vegetation Index (senal de clorofila) SAVI — Soil-Adjusted Vegetation Index BSI — Bare Soil Index (ruido — peso negativo) Los pesos son optimizados por ML para cada tipo de cultivo y zona climatica
Biochar Fresia WalterLand
FOR.01 // FHI
Monitoreo Forestal — Salud del Bosque
Monitoreo a gran escala de bosques tropicales. Combina vitalidad y humedad para deteccion temprana de degradacion y estres hidrico.
FHI = (w_ndvi x NDVI) + (w_ndmi x NDMI) Calibracion Amazonia: w_ndvi = 0.6 w_ndmi = 0.4 Deteccion de cambios: ΔFHI = FHI_actual - FHI_base Alerta de degradacion: FHI < umbral critico por 2 meses consecutivos
Amazonia Peumayen
FOR.02 // RF-SPECIES
Especie Invasora — Clasificador Probabilistico
Mapeo de probabilidad de presencia de Ulex europaeus usando Random Forest multi-indice. Genera mapa 0-100% en vez de mascara binaria.
P(especie) = RandomForest(NDVI, NDMI, YFI, SAVI, NBR2, NDRE) 6 features espectrales —> modelo entrenado —> probabilidad [0, 1] YFI = Yellow Flower Index (firma espectral de floracion)
Ulex
GEO.01 // BTI
Basalt Target Index
Deteccion espectral de afloramientos basalticos en ambientes humedos para cuantificacion del potencial de Enhanced Rock Weathering (ERW). Identifica roca basaltica pura eliminando vegetacion y agua.
BTI = (w_mi x MI) + (w_fi x FI) - (w_ndvi x NDVI) - (w_mndwi x MNDWI) MI (Mafic Index) = (SWIR1 - SWIR2) / (SWIR1 + SWIR2) minerales Fe+Mg FI (Ferrous Iron) = SWIR1 / NIR oxidos de hierro NDVI = (NIR - Red) / (NIR + Red) supresion veg. MNDWI = (Green - SWIR1) / (Green + SWIR1) eliminacion agua w_mi = 2.5 w_fi = 1.8 w_ndvi = 1.5 w_mndwi = 1.2 Bandas: B3 (Green), B4 (Red), B8 (NIR), B11 (SWIR1), B12 (SWIR2)
Resultados ERW Chile: 47,439 ha detectadas (bruto) | ~309 Mt CO2e potencial | 1.43M ha analizadas
Ventaja clave: Elimina falsos positivos en zonas humedas de Patagonia donde indices estandar fallan por alta cobertura vegetal.
ERW Chile
GEO.02 // KRIGING
Interpolacion Geoestadistica
Extension del framework a datos puntuales mediante Kriging ordinario. Genera superficies continuas de contaminantes atmosfericos a partir de estaciones de monitoreo discretas.
Z(s₀) = Σ λᵢ x Z(sᵢ) Variograma ajustado por modelo esferico Validacion cruzada: LOOCV con RMSE Implementacion: R (gstat, sf, terra)
NO2 Santiago
